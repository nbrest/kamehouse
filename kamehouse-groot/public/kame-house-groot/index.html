まだまだだね


KameHouse GRoot is a small app built to do admin tasks independently of having all the KameHouse modules deployed on tomcat

This app runs on an plain http server such as apache httpd and its few API's are built with php. The APIs are mainly to get the current httpd session status and to execute scripts in the server (from my.scripts private repo) . It's main functionality is to manage the tomcat running /kame-house to start/stop/deploy/undeploy it's modules, pull from my git repos and update my media files on the media server. It can also trigger all those commands in all servers with one click connecting to the KameHouse GRoot API's on each server

Most of the functionality is in kame-house. This app is just a UI to do all the admin stuff I usually do through the command line
