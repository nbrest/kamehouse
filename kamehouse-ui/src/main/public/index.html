What is KameHouse

KameHouse is a set of webapps and command line tools written in Java, JavaScript and Bash that can be used together to execute the functionality mentioned below

Other things you can do with KameHouse

  • Manage all media files and play them with VLC
  • Schedule automatic bookings of cardio and comps on TennisWorld
  • Execute all test functionality with the test module
  • Manage all KameHouse entities and users
  • Manage the backend settings like ehcache, log level and job scheduler
  • Schedule server restarts
  • Wake up media server
  • Restart httpd running on the current server
