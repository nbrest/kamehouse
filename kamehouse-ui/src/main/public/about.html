About KameHouse

KameHouse is a project I started mainly to practice Java and frontend development. Initially it was only meant for managing VLC player. But later on I added a lot more functionality
Image-Links
Image-Links
GitHub
KameHouse is an open source project. You can find all the source code in GitHub
DockerHub
KameHouse is hosted on DockerHub! The easiest way to run it is using KameHouse's docker container

KameHouse is useful to ...

  • Build and manage your media library of music and videos
  • Control a server running VLC player attached to a TV
  • Book courts and lessons on tennis world
  • Wake kamehouse-r2d2 with Wake On Lan functionality
  • Deploy updates of KameHouse modules in the current server
  • A lot more... The whole project description is on GitHub
